O que você vai aprender neste curso

1. Regras da ABNT

  1. Conhecendo a ABNT
  2. ABNT para os elementos pré-textuais
  3. ABNT para os elementos textuais
  4. ABNT para os elementos pós textuais

2. Estruturação de resumos e fichamentos

  1. O que é um resumo e sua importância
  2. Tipos de resumo
  3. Como elaborar os resumos
  4. O que é um fichamento e sua importância
  5. Tipos de fichamento
  6. Como elaborar os fichamentos

3. Estruturação da monografia

  1. Definindo tema e título
  2. Pensando na problemática e pergunta de pesquisa
  3. Definindo os objetivos(geral e específico)
  4. Elaborando os elementos pré-textuais
  5. Elaborando a introdução
  6. Elaborando o desenvolvimento
  7. Elaborando as conclusões
  8. Elaborando as referências e demais elementos pós textuais

4. Estrutura do artigo

  1. Elaborando os elementos pré-textuais
  2. Elaborando a introdução
  3. Elaborando o desenvolvimento
  4. Elaborando as conclusões
  5. Elaborando as referências
